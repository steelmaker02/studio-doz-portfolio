# Studio DOZ Portfolio

Minimalistisches Fotoportfolio für Studio DOZ. Ein zeitgenössisches Projekt, das sich auf visuelle Klarheit, Raum und Reduktion konzentriert.

## 🚀 Technologien
- Vite
- Tailwind CSS
- Vanilla JavaScript
- Hosted on GitHub Pages

## ✨ Features
- Minimalistisches Design mit Fokus auf Typografie.
- Responsive Bilder-Galerie mit Kategorien.
- Cinematic Reveal Animationen.
- Lichtbox-Modus für Bilder.
- Kontaktformular mit Modal-Integration.
- SEO-optimiert.
